从上到下，不同表格的优先级按从高到低排列，其中，同一个表格中的多个运算符具有相同的优先级

结合律运算符功能用法
左::全局作用域::name
左::类作用域class::name
左::命名空间作用域namespace::name
结合律运算符功能用法
:-::-::-::-:
左.成员选择object.member
左->成员选择pointer->member
左[]下标expr1[expr2]
左()函数调用name(expr_list)
左()类型构造type(expr_list)
结合律运算符功能用法
:-::-::-::-:
右++后置递增运算lvalue++
右--后置递减运算lvalue--
右typeid类型 IDtypeid(type)
右typeid运行时类型 IDtypeid(expr)
右explicit cast类型转换cast_name<type>(expr)
结合律运算符功能用法
:-::-::-::-:
右++前置递增运算++lvalue
右--前置递减运算--lvalue
右~位求反~expr
右!逻辑非!expr
右-一元负号-expr
右+一元正号+expr
右*解引用*expr
右&取地址&expr
右()类型转换(type) expr
右sizeof对象的大小sizeof expr
右sizeof类型的大小sizeof(type)
右sizeof...参数包的大小sizeof...(name)
右new创建对象new type
右new[]创建数组new type[size]
右delete释放对象delete expr
右delete[]释放数组delete[] expr
右noexcept能否抛出异常noexcept (expr)
结合律运算符功能用法
:-::-::-::-:
左->*指向成员选择的指针ptr->*ptr_to_member
左.*指向成员选择的指针obj.*ptr_to_member
结合律运算符功能用法
:-::-::-::-:
左*乘法expr1 * epxr2
左/除法expr1 / epxr2
左%取模（取余）expr1 % epxr2
结合律运算符功能用法
:-::-::-::-:
左+加法expr1 + epxr2
左-减法expr1 - expr2
结合律运算符功能用法
:-::-::-::-:
左<<向左移位expr1 << expr2
左>>向右移位expr1 >> expr2
结合律运算符功能用法
:-::-::-::-:
左<小于expr1 < expr2
左<=小于等于expr1 <= expr2
左>大于expr1 > expr2
左>=大于等于expr1 >= expr2
结合律运算符功能用法
:-::-::-::-:
左==相等expr1 == expr2
左!=不相等expr1 != expr2
结合律运算符功能用法
:-::-::-::-:
左&位与expr1 & expr2
结合律运算符功能用法
:-::-::-::-:
左^位异或expr1 ^ expr2
结合律运算符功能用法
:-::-::-::-:
左|位或expr1 | expr2
结合律运算符功能用法
:-::-::-::-:
左&&逻辑与expr1 && expr2
结合律运算符功能用法
:-::-::-::-:
左||逻辑或expr1 || expr2
结合律运算符功能用法
:-::-::-::-:
右? :条件expr1 ? expr2 : epxr3
结合律运算符功能用法
:-::-::-::-:
右=赋值lvalue = expr
右*=,/=, %=, +=, -=, <<=, >>=, &=, |=, ^=复合赋值lvalue += expr 等
结合律运算符功能用法
:-::-::-::-:
右throw抛出异常throw expr

:-::-::-::-:
左,逗号expr1, expr2
